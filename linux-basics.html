Linux Basics

Essential commands, navigation, file management, and utilities.

pwd — Print Working Directory
Shows the full path of the folder you are currently in.
Syntax: pwd
Example: /home/user/Documents
cd — Change Directory
Move into another folder.
Syntax: cd <directory>
Examples: cd Documents # go into the Documents folder cd .. # go one folder back (parent folder) cd ~ # go to your home directory

ls — List Directory Contents
Lists files and folders in the current directory. Use -al to see hidden files and detailed info (permissions, size, owner).
Syntax: ls ls -al
Examples: ls # basic list ls -al # detailed list including hidden files
cat — View File Contents
Displays the contents of a file directly in the terminal.
Syntax: cat <filename>
Example: cat test.txt
man — Manual / Help
Opens the built-in manual page for any command. Press q to exit.
Syntax: man <command>
Example: man ls # opens the manual for ls

mkdir — Make Directory
Creates a new folder.
Syntax: mkdir <foldername>
Example: mkdir projects
rmdir — Remove Directory
Deletes an empty folder.
Syntax: rmdir <foldername>
Example: rmdir projects
echo — Write Text to a File
Prints text to the terminal, or use > to write it into a file (creates the file if it doesn't exist).
Syntax: echo 'your text' > filename.txt
Example: echo 'hello world' > test.txt
cp — Copy
Copies a file to another location.
Syntax: cp <filename> <destination>
Example: cp test.txt /home/user/backup/
mv — Move
Moves a file to another location. Also used to rename files.
Syntax: mv <filename> <destination>
Example: mv test.txt /home/user/Documents/
rm — Remove / Delete
Permanently deletes a file. No recycle bin — be careful!
Syntax: rm <filename>
Example: rm test.txt

passwd — Change Password
Changes the password for your current user account. It will prompt you for your current password, then ask for the new one.
Syntax: passwd
sudo apt update && locate — Update & Search
sudo apt update refreshes the package list.
locate searches your system for a file by name.
The && means "run the second command only if the first one succeeds."
Syntax: sudo apt update && locate <filename>
Example: sudo apt update && locate passwords.txt

Ctrl + L
Clear the terminal screen (same as typing the command: clear)

explainshell.com
→ Paste any command here to get a clear breakdown of what every part of it does. Great for commands you don't recognise.
man <command>
→ Always your first stop for help. Built into Linux.
Example: man cp
